Config files managed with [GNU Stow].

Working on [Nushell] scripting.

```nushell
# update all configs
$ nu stow.nu all

# delete all configs
$ nu stow.nu delete
```

https://venthur.de/2021-12-19-managing-dotfiles-with-stow.html
